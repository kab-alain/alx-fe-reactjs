GitHub User Search

A React application that allows users to search for GitHub profiles and view basic user information. Built with Vite, React, and Axios.

Features

Search for GitHub users by username.

View user details including:

Avatar

Username

Profile link

Responsive design for desktop and mobile.

Prepared for API integration using environment variables.

Tech Stack

React (with Vite)

Axios (for API requests)

JavaScript (ES6+)

CSS (or your preferred styling)

GitHub API
